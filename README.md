#  Task Manager (React + Spring Boot + JWT)

A full-stack **Task Management System** built with **Spring Boot** and **React**, featuring **JWT Authentication** and complete **Task CRUD** operations with a minimal responsive UI.

---

##  Features

###  Authentication (JWT)
- User Register
- User Login
- JWT Token generation
- Protected routes (only authenticated users can access dashboard)
- Logout

###  Task Management
- Create Task
- List All Tasks (only logged-in user tasks)
- Get Task by ID (Search)
- Update Task
- Delete Task
- Update Task Status (TODO / IN_PROGRESS / DONE)

###  UI
- Minimal clean UI with **Tailwind CSS**
- Navbar with:
  - Logo
  - Search bar
  - Logged-in user first letter avatar
  - Logout button

---

##  Tech Stack

### Backend
- Spring Boot
- Spring Security
- JWT Authentication
- Spring Data JPA
- MySQL (or H2)
- Lombok

### Frontend
- React (Vite)
- Axios
- React Router DOM
- Tailwind CSS
